burpsuite优化配置

  • burpsuite
tools 
burpsuite是由PortSwigger公司开发的一款功能强大的web应用程序测试工具．
它不同于一般的web扫描器，它不局限与对网站的扫描，理论上，他可以测试一切通过http协议通讯的程序．

安装设置中文字体

  • 安装字体

    百度＂文泉驿微米黑＂下载一个（微软雅黑因该也可以，文泉驿是苑老师推荐的），扔到home目录下的.font目录里就行了．

  • 设置中文字体

    设置的位置：User options -> Dispaly -> HTTP Message Display
    点击　change font … 按钮　修改　Font 属性，选择要设置的字体即可．

设置较大的JVM内存

burpsuite是个java程序，如果安装了jre就能顺利运行了，但是由于burpsuite是个做渗透测试的工具，免不了发送大量请求，那么默认的64M内存也就满足不了需求了．所以就要在java在创建虚拟机的时候分配较大的内存，命令如下：

java -jar -Xmx2048M  /your_burpsuite_path/burpsuite.jar

其中参数-Xmx指定JVM可用的最大内存，单位可以是M，也可以是G，如果是G为单位的话，则脚本内容为：

java -jar -Xmx2G  /your_burpsuite_path/burpsuite.jar
